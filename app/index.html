Comment ça fonctionne

L'outil lit passivement la zone du log de combat visible à l'écran grâce à la reconnaissance optique de caractères (OCR). Il ne touche pas à la mémoire ni aux processus du jeu.


1. Configurez la zone (résolution du jeu suffisant dans 99% des cas)

2. Cliquez Démarrer avant ou pendant une session

3. Les dégâts s'accumulent en temps réel

4. Cliquez Arrêter pour exporter en CSV

Installation (première fois)
# Python 3.10 ou plus récent requis
pip install flask pillow pytesseract numpy opencv-python mss

# Tesseract OCR (moteur de lecture)
# Windows : https://github.com/UB-Mannheim/tesseract/wiki
# macOS   : brew install tesseract
# Linux   : sudo apt install tesseract-ocr
Activer le log de combat en jeu
Échap → Paramètres → Interface
Activer : "Afficher les dégâts du joueur"
Ajuster la taille et la transparence selon vos préférences.
